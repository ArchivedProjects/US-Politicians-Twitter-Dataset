# US Politicians Twitter Dataset
 Dataset based on usernames of American politicians. Data extracted from Wikidata.
 
 Variables : 
 1. Mandatory

    a. Name

    b. Twitter username

    c. Sex

    d. Birth

2. Optional

    a. Instagram
    
    b. Political party
    
    c. Image

 CSV : 2247 rows.
 Last update: November 16, 2020.
